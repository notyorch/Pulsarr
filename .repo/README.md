# .repo

This folder contains repository-level metadata and operational guidance used by maintainers and contributors.

Files:

- `SECURITY_OVERVIEW.md` — high-level summary of security settings for this repository.
- `SECURITY.md` — instructions for how users should report security vulnerabilities.
- `DEPENDENCY_AND_SCANNING.md` — notes about Dependabot, code scanning and secret scanning status and recommended next steps.

Place any other repo-level governance or operational documents here (e.g., CONTRIBUTING, CODE_OF_CONDUCT, release procedures).
